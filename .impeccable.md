# Impeccable Design Context — 听见北京

## Design Context

### Users
- 没来过北京的人，想通过声音和视觉感受这座城市
- 来过北京的人，想重新发现它不只有宏大，还有细小真实的温度
- MTX AI Coding 营的导师和同学（展示场景，1 分钟内需理解核心玩法）
- 使用情境：在电脑或手机浏览器中打开，滚动浏览，点击互动

### Brand Personality
- 克制 (Restrained) — 不喧哗，不堆砌，留白即表达
- 高级 (Refined) — 数字展览感，非旅游宣传页
- 有温度 (Warm) — 通过声音和文案传递城市的呼吸感

三字概括：**克制 · 高级 · 有温度**

情感目标：让用户在浏览结束时，感觉"听见过"一座城市，而不只是"看过了"一个网页。

### Aesthetic Direction

**视觉基调**：极简编辑风 × 东方秩序感 × 数字展览
- 灵感来源：日本设计中白 (Kenya Hara) 的留白哲学、中国古籍排版的竖向秩序、现代艺术展览的策展逻辑
- 参考气质：博物馆数字策展、独立杂志排版、Apple 产品页的克制感

**反参考 (Anti-references)**：
- ❌ 廉价国风素材（水墨毛笔字、龙凤图案、红色灯笼满屏）
- ❌ 旅游宣传页式布局（大图轮播 + 推荐路线 + 餐厅列表）
- ❌ AI 泛化审美（紫蓝渐变、霓虹点缀、毛玻璃滥用、居中一切）
- ❌ 花哨 3D、大面积渐变、弹跳动画

**主题**：Light mode（米白底），宫墙红为唯一强调色

### Design Principles

1. **声音是主线，不是装饰** — 所有交互围绕"听见"展开，视觉为声音服务
2. **留白即内容** — 不怕空，空间本身就是表达；每段文字都有存在的理由
3. **一个强调色原则** — 宫墙红 #8B2E2E 只用于按钮、进度、当前选中、小编号，不滥用
4. **动效克制** — 文字淡入、卡片轻浮、进度增长，用指数缓动而非回弹；状态变化有意义
5. **东方秩序** — 编号系统 (01/02/03) 建立层次；中轴线作为视觉隐喻贯穿页面

### Color System
```css
--bg: #F6F3EE;        /* 米白底，如旧纸 */
--text: #111111;       /* 深黑正文 */
--muted: #666666;      /* 次级灰 */
--line: #D8D3C8;       /* 分割线 */
--red: #8B2E2E;        /* 宫墙红，唯一强调色 */
--gold: #B89B5E;       /* 金色，极少量点缀 */
```
使用规则：背景 80% 米白，正文黑灰，红色占比 <5%，金色 <1%

### Typography
```css
/* 标题：衬线，文学感 */
font-family: "Noto Serif SC", "Songti SC", serif;

/* 正文：无衬线，现代清晰 */
font-family: "Inter", "Noto Sans SC", "PingFang SC", sans-serif;
```
- Hero 标题：clamp(48px, 8vw, 96px)
- Section 标题：clamp(32px, 5vw, 56px)
- 正文：16px，行高 1.8
- 正文宽度：max-width 680px
- 编号使用大号衬线，作为视觉锚点

### Motion
- Section 进入：文字 fade-in + slight translateY，ease-out-quart
- 卡片 hover：translateY(-4px) + 轻微 brightness
- 进度条：width 过渡，500ms ease-out
- 声音播放：卡片边缘出现细线高亮
- 背景：极淡的中轴线抽象线条（SVG，opacity < 0.05）
- ❌ 不做：弹跳、大面积渐变、3D 翻转、火花线
